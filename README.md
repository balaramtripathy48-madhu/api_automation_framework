API Framework

A beginner API automation testing framework built using Python, Pytest, Requests, SQL, Git, and GitHub.

Features

- GET API testing
- POST API testing
- PUT API testing
- DELETE API testing
- Reusable API functions
- Pytest fixtures
- Parameterized tests
- Basic database validation

Technologies Used

- Python
- Pytest
- Requests
- MySQL
- Git
- GitHub

Project Structure

API_FRAMEWORK/
├── all_api/
├── test_api/
├── conftest.py
├── requirements.txt
└── README.md

Installation

pip install -r requirements.txt

Run Tests

python -m pytest -v

Test Coverage

- Status code validation
- Response validation
- CRUD operation testing
- Basic database checks

Purpose

Created for practicing API automation testing and improving QA skills.
